OdinSync

Synchronize. Automate. Grow.

OdinSync is an open-source, AI-ready Business Operating System for SMEs.

It will start as a modular monolith using Java and Spring Boot, then evolve into microservices with Kafka, Kubernetes, CI/CD, and AI capabilities.

Current Status:

Phase 0 – Project Foundation

Architecture:

Modular Monolith first, Microservices later.

Main goals:

- Multi-tenant SaaS

- Domain-Driven Design

- Clean Architecture

- Spring Boot backend

- Future Kafka and Kubernetes support

- Future AI Copilot
