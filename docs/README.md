# OdinSync Documentation

This folder contains product, architecture, ADR, API, database, deployment, development, and runbook documentation.

## Main Sections

- product/: product vision and business requirements
- architecture/: system design and DDD documentation
- adr/: architecture decision records
- api/: API documentation
- database/: schema and migration documentation
- deployment/: Docker, Kubernetes, and cloud deployment
- development/: coding standards and contribution workflow
- runbooks/: operational guides
